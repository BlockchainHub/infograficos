O Básico de uma DAO na Rede Ethereum

O Que Significa DAO?

DAO (Decentralized Autonomous Organization) pode ser traduzida como Organização Autônoma Descentralizada.

  • Organização
  • Existe com o intuito de atingir fins em comum.
  • Autônoma
  • É capaz de decidiv e agir de forma autônoma de acordo com suas regras programadas.
  • Descentralizada
  • Não depende de uma infra estrutura central.

O Que É uma DAO?

Dois fatores são essenciais para formar uma DAO na Ethereum:

  • Um código de smart contracts é escrito para definir as regras da organização

  • O código é implantado na blockchain da plataforma Ethereum.

Dois fatores são essenciais para formar uma DAO na Ethereum:

  • Os participantes mantêm controle direto em tempo real dos fundos contribuídos.
  • Regras de governança são formalizadas, automatizadas e forçadas por meio de software.

Como Funcinoa uma DAO?

A DAO necessita de “ether” para funcionar.
